## 👥 スタッフ運用設定

### 3. スタッフ管理方針

**Q1: スタッフは何名いますか？**
- [ ] 1名（オーナーのみ） → シンプル設定推奨
- [ ] 2-5名 → 標準設定
- [ ] 6名以上 → シフト管理推奨

**Q2: スタッフ指名機能は必要ですか？**
- [ ] **必要** → `enableStaffSelection = true`
  - 例: 美容室、整体院、パーソナルトレーニング
- [ ] **不要** → `enableStaffSelection = false`
  - 例: 飲食店（席があれば誰でもOK）、カフェ

**Q3: スタッフ指名は必須ですか？（Q2で「必要」の場合のみ）**
- [ ] **必須**（必ずスタッフを選択） → `requireStaffSelection = true`
  - 例: スタイリスト指名制の美容室
- [ ] **任意**（「指名なし」も選択可能） → `requireStaffSelection = false`
  - 例: 担当者不問のクリニック

**Q4: スタッフのシフト管理は必要ですか？**
- [ ] **必要** → `enableStaffShifts = true`
  - スタッフごとに勤務日・勤務時間を設定
  - 休暇管理が必要
- [ ] **不要** → `enableStaffShifts = false`
  - 全員が常に勤務している想定（個人店向け）

## 📅 予約フロー設定

### 6. 予約受付期間

**Q6: 何日前から予約を受け付けますか？**
- [ ] 当日から受付可能 → `minAdvanceBookingDays = 0`
- [ ] 翌日以降のみ受付 → `minAdvanceBookingDays = 1`
- [ ] その他: _____ 日後から → `minAdvanceBookingDays = _____`

**Q7: 何日後まで予約を受け付けますか？**
- [ ] 1ヶ月先まで → `maxAdvanceBookingDays = 30`
- [ ] 3ヶ月先まで → `maxAdvanceBookingDays = 90`
- [ ] 6ヶ月先まで → `maxAdvanceBookingDays = 180`
- [ ] その他: _____ 日後まで → `maxAdvanceBookingDays = _____`

### 7. 予約承認制

**Q8: 予約は即座に確定しますか？それとも管理者承認が必要ですか？**
- [ ] **即座に確定** → `requireApproval = false`
  - 顧客が予約した瞬間に確定
  - 例: ファストカジュアル、美容室
- [ ] **管理者承認が必要** → `requireApproval = true`
  - 管理者が承認するまで「予約リクエスト」状態
  - 例: 高級レストラン、席調整が必要な店舗

### 8. キャンセル期限

**Q9: 予約日の何時間前までキャンセルを受け付けますか？**
- [ ] 2時間前まで → `cancellationDeadlineHours = 2`
- [ ] 24時間前まで（前日まで） → `cancellationDeadlineHours = 24`
- [ ] 48時間前まで（2日前まで） → `cancellationDeadlineHours = 48`
- [ ] 1週間前まで → `cancellationDeadlineHours = 168`
- [ ] その他: _____ 時間前まで → `cancellationDeadlineHours = _____`

**理由**: 直前キャンセルを防ぎ、機会損失を減らすため

## 📧 メール通知設定

### 9. 通知メール

**Q10: どの通知メールを送信しますか？**

- [ ] **予約確定メール** （推奨: ON）
  - 顧客が予約した際に確認メールを送信
- [ ] **予約キャンセル通知** （推奨: ON）
  - キャンセル時に顧客に通知
- [ ] **前日リマインダー** （推奨: ON）
  - 予約前日にリマインダーを送信
- [ ] **当日リマインダー** （推奨: OFF）
  - 予約当日の朝にリマインダーを送信
- [ ] **管理者への新規予約通知** （推奨: ON）
  - 新規予約が入った際に店舗管理者にメール通知

**設定値（JSON）**:
```json
{
  "reservationConfirmed": true,
  "reservationCancelled": true,
  "reminderBeforeDay": true,
  "reminderSameDay": false,
  "adminNotification": true
}
```
